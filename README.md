# Login Form

Login Form Built Using HTML, CSS and JavaScript.

![Login Form Screenshot](https://raw.githubusercontent.com/BinaryVortex/Login-Form-38/main/Screenshot%202024-07-03%20085032.png)

## Overview

A clean, minimal login form created with plain HTML, CSS, and JavaScript. This repository contains a single-page implementation (index.html) and the styling (style.css). It's easy to use, customize, and integrate into small projects or learning demos.

## Features

- Simple and elegant UI
- Lightweight (no frameworks)
- Easy to customize styles
- Basic client-side validation (email/username and password fields)

## Files

- `index.html` — The login form markup and JavaScript.
- `style.css` — Styling for the login form.
- `Screenshot 2024-07-03 085032.png` — Preview image used in this README.

## How to use

1. Clone the repository:

   git clone https://github.com/BinaryVortex/Login-Form-38.git

2. Open `index.html` in your web browser:

   - Double-click `index.html` or
   - Run from a local server (recommended for testing any JavaScript behavior):

     python -m http.server 8000

     Then visit http://localhost:8000 in your browser.

## Customization

- Edit `style.css` to change colors, spacing, and layout.
- Modify `index.html` to add or remove form fields.
- Enhance validation or connect to a backend by updating the JavaScript inside `index.html`.
